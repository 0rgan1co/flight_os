# FlightOS

Dashboard personal de gestión estratégica en tres niveles:

- **N1 — Estratégico:** Visión, North Star Metric, Outcomes e Input Metrics
- **N2 — Coordinación:** Iniciativas con Ecocycle (Semilla / Brote / Crecimiento / Compost)
- **N3 — Ejecución:** OST (Oportunidades → Soluciones → Experimentos) + Revisión semanal

Datos persistidos en Supabase. Acceso vía GitHub Pages.
